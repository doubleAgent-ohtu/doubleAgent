### 1. Introduction

**Double Agents** is an AI-powered chatbot application where two AI bots (Chatbot A and Chatbot B) converse with each other based on an initial message provided by the user. You can customize bot behavior with system prompts and follow the conversation in real-time.

---

### 2. Logging In

1. Open the application in your browser.
2. You will see a login page with a **"Log in with university credentials"** button.
3. Click the button and log in with your University of Helsinki credentials.
4. After successful login, you will be redirected to the application's home page.

---

### 4. Setting System Prompts

A system prompt defines how the chatbot behaves in the conversation.

1. Click the **"Set Prompt"** or **"Edit Prompt"** button in either Chatbot A's or B's box.
2. In the dialog that opens:
   - Enter your desired system prompt in the **System Prompt** field.
   - *Optional:* Give the agent a name in the **Agent Name** field if you want to save the prompt.
3. Choose an action:
   - **Set** â€“ Sets the prompt for this session
   - **Set & Save** â€“ Sets the prompt and saves it to the database for future use

**Example system prompt:**
```
You are a philosopher who ponders life's big questions.
Always respond thoughtfully and with questioning.
```

### 5. Starting a Conversation

1. Type a starting message in the **"Conversation starter..."** field.
2. Select your desired settings:
   - **Model** â€“ Choose the AI model to use (GPT-4o, GPT-4o-mini, GPT-4.1, GPT-5)
   - **Turns** â€“ Set the number of conversation turns (1â€“10)
3. Click the **"Start"** button.
4. The bots will begin conversing with each other. Messages will appear in the conversation area in real-time.

---

### 6. Stopping a Conversation

If you want to stop the conversation before all turns are completed:

1. Click the red **"STOP"** button.
2. The conversation will stop immediately.

---

### 7. Model Selection

The application supports the following models:

| Model | Description |
|-------|-------------|
| **GPT-4o** | Default choice, balanced performance |
| **GPT-4o-mini** | Lighter and faster version |
| **GPT-4.1** | More advanced version |
| **GPT-5** | Latest model |

Select a model from the dropdown menu before starting a conversation.

### 10. Troubleshooting

| Problem | Solution |
|---------|----------|
| Cannot log in | Check your university credentials and network connection |
| Conversation won't start | Make sure you have entered a starting message |
